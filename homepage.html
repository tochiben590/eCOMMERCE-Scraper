Welcome to my eCOMMERCE Scrapper API

You can now easily get the prices and details of products from the top eCOMMERCE store 'AMAZON' in one place

Here are the detailed use case of its ENDPOINT;


ENDPOINT Description Use-Case

/amazon/search/:product name

Logs out different JSON results of the product you are querying, that is the PRODUCT NAME, PRICE, and even DESCRIPTION

A typical example is

/amazon/search/Apple Iphone 14 pro max

/amazon/ASIN

ASIN: which is a product id in Amazon, it usually starts with uppercase B in an amazon product url

It always look like this B0BGYFW73W

Logs out full details of a product in JSON format

A typical example is

/amazon/B0BGYFW73W

/amazon/ASIN/reviews

Logs out reviews of the product in JSON format

A typical example is

/amazon/B0BGYFW73W/review
